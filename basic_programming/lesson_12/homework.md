### Task 1 "Grades"

Create an array of 5 elements to store a student's grades. Fill the array with grades and calculate the student's
average score.

- Use Scanner to fill the array.

### Task 2 Remove Element

Write a program that removes an element from the array based on conditions:

- by a given position.
- by value

### Task 3 Frequency Analysis

The program should count how many times each number appears in the array and display the result in the console.

### Task 4 Number of Occurrences

The user enters two numbers: the length of the array and the number X. The program fills the array with random numbers (
the number X may not be in the array).
**Task** — find out how many times the number X appears in the array.

### Task 5 Uniqueness Check

The program should check whether all elements in the array are unique.

### Задача 1 "Оценки"

Создайте массив из 5 элементов для хранения оценок студента. Заполните массив оценками и вычислите средний балл
студента.

- используйте Scanner для заполнения массива.

### Задача 2 Удаление элемента

Напишите программу, которая удаляет из массива элемент по условиям:

- по заданной позиции.
- по значению

### Задача 3 Частотный анализ

Программа должна подсчитать, сколько раз каждое число встречается в массиве и вывести результат в консоль.

### Задача 4 Число встреч

Пользователь вводит два числа: длину массива и число X. Программа заполняет массив случайными числами (число X может не
оказаться в масиве).
**Задача** — найти, сколько раз число X встречается в массиве.

### Задача 5 Проверка на уникальность

Программа должна проверять, все ли элементы в массиве уникальны.
